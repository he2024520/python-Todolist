# TodoList
一个使用Python编写命令行任务管理程序

## 功能

✅ 添加任务
✅ 删除任务
✅ 修改任务状态
✅ 修改任务名称
✅ 关键词搜索任务
✅ 自动保存任务（JSON）

## 使用方法
运行文件 'task_file2.py'

## 使用技术
- Python
- JSON
- 文件读写

